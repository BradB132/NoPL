This document describes the makeup of a NoPL compiled binary file. A compiled binary file
consists of zero or more statements, each will have at least one operator, as detailed
below.  Each operator is detailed in byte size and order below

///////////////////////////////////////
//BASIC TYPES AND BYTE SIZES

NoPL_Index - unsigned, 2 bytes,
used wherever there is a subscript or list of items

NoPL_Instruction - unsigned, 1 byte,
used for distinguishing different operators

NoPL_BufferMove - signed, 4 bytes,
used for jumping immediately to a different part of the program, like a goto statement

///////////////////////////////////////
